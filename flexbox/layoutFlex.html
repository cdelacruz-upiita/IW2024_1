El modelo de Caja Flexible, comúnmente llamado Flexbox, fue diseñado como un modelo unidimensional de layout, y como un método que pueda ayudar a distribuir el espacio entre los elementos de una interfaz y mejorar las capacidades de alineación.

El termino unidimensional se refiere a que el Layout se organiza en una sola dimensión a la vez (eje).

Ejes en Flexbox

  1. Eje principal definido por la propiedad flex-direction.

    2. Posibles valores de flex-direction: row, row-reverse, column, column-reverse. Row y row-reverse indican que el eje principal es horizontal. Column y column-reverse establecen que el eje principal es vertical.

  2. Eje cruzado perpendicular al eje principal.

    3. Este eje va en sentido opuesto al eje principal, es decir, si el eje principal es horizontal el eje cruzado será vertical.

Lineas de inicio y fin

Se asocia las lineas de inicio y fin de una caja flexible al modelo de escritura, es decir, izquierda - derecha, derecha-izquierda de acuerdo con el idioma seleccionado para el contenido.

Contenedor Flexbox

Los elementos de un contenedor Flex se comportan por defecto de la siguiente manera:

  1. Los elementos se despliegan sobre una fila (valor por defecto de flex-direction es row).
  2. Los elementos empiezan desde el margen inicial del eje principal.
  3. Los elementos no se ajustarán en la dimensión principal, pero se pueden contraer.
  4. Los elementos se ajustarán para llenar el tamaño del eje cruzado.
  5. La propiedad flex-basis es definida como auto.
  6. La propiedad flex-wrap es definida como nowrap.
